The callbacks

About the callbacks

The callbacks are functions called by B3DMM to allow the user extends the cheat behavior.

Usage

b3dmmAddCallback(callback_id, callback_function)

List

  • CALLBACK_AT_RENDERING_BACKGROUND
  • CALLBACK_AT_RENDERING_INTERFACE
  • CALLBACK_AT_RENDERING_PIE_MENU
  • CALLBACK_AT_DESTROY_SCRIPT
  • CALLBACK_AT_HUD_REDRAW
  • CALLBACK_AT_CL_CREATEMOVE

CALLBACK_AT_RENDERING_BACKGROUND

This function will be called to perform rendering at the background scene (behind all the menus). You should not use this function to render using the game engine API here.

Examples

Lua
function atRenderingBackground()
    -- Your render functions goes here...
end

b3dmmAddCallback(CALLBACK_AT_RENDERING_BACKGROUND, atRenderingBackground)
JavaScript
function atRenderingBackground() {
  // Your render functions goes here...
}

b3dmmAddCallback(CALLBACK_AT_RENDERING_BACKGROUND, atRenderingBackground);

CALLBACK_AT_RENDERING_INTERFACE

This function will be called to perform rendering at the top scene (menus layout). You should not use this function to render using the game engine API here.

Examples

Lua
function atRenderingInterface2D()
    -- Your guiWindow functions goes here...
end

b3dmmAddCallback(CALLBACK_AT_RENDERING_INTERFACE, atRenderingInterface2D)
JavaScript
function atRenderingInterface2D() {
  // Your guiWindow functions goes here...
}

b3dmmAddCallback(CALLBACK_AT_RENDERING_INTERFACE, atRenderingInterface2D);

CALLBACK_AT_DESTROY_SCRIPT

Not used currently.

CALLBACK_AT_HUD_REDRAW

Called by the game when the Head Ups Display function is being called. You should only use this function to render using the game engine API here.

Examples

Lua
function atHudRedraw(time, intermission)
    -- Your guiWindow functions goes here...
end

b3dmmAddCallback(CALLBACK_AT_HUD_REDRAW, atHudRedraw)
JavaScript
function atHudRedraw(time, intermission) {
  // Your guiWindow functions goes here...
}

b3dmmAddCallback(CALLBACK_AT_HUD_REDRAW, atHudRedraw);

CALLBACK_AT_CL_CREATEMOVE

Called by the game when the client needs to process a user command.

Examples

Lua
function atClientCreateMove(frametime, cmd, active)
    -- Your physics code for the local player goes here...
end

b3dmmAddCallback(CALLBACK_AT_CL_CREATEMOVE, atClientCreateMove)
JavaScript
function atClientCreateMove(frametime, cmd, active) {
  // Your physics code for the local player goes here...
}

b3dmmAddCallback(CALLBACK_AT_CL_CREATEMOVE, atClientCreateMove);
